bitbeanrpc.config — Utilities for reading bitbean configuration files¶

Utilities for reading bitbean configuration files.

bitbeanrpc.config.read_config_file(filename)¶

Read a simple '='-delimited config file. Raises IOError if unable to open file, or ValueError if an parse error occurs.

bitbeanrpc.config.read_default_config(filename=None)¶

Read bitbean default configuration from the current user’s home directory.

Arguments:

  • filename: Path to a configuration file in a non-standard location (optional)
